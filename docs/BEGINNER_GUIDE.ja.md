# Claude Code Workbench やさしい解説

このページは、Claude Code Workbench を中学生でも分かるように説明するページです。

## このプロジェクトは何か

Claude Code Workbench は、Claude Code CLI を1つのターミナル画面の中に複数並べて使うための道具です。

1つのAIに全部を考えさせるのではなく、画面ごとに役割を分けられます。

たとえば:

- 1つ目の画面は、テーマを調べる
- 2つ目の画面は、別のアイデアを出す
- 3つ目の画面は、実装計画を考える
- 4つ目の画面は、危ない点を確認する
- 5つ目の画面は、最後にまとめる

## 何のために役立つか

- 調査を複数方向から進める
- アイデアを比べる
- 実装前に計画を立てる
- リスクを確認する
- 結果を Markdown で残す

## たとえ話

グループ学習を考えてください。

1人が調べ、1人がノートを書き、1人が間違いを探し、1人が最後にまとめます。

Claude Code Workbench は、Claude Code CLI にそのような役割分担をさせるための作業机です。

## 注意点

この道具は、GitHubの画面上でそのまま動くアプリではありません。  
自分のMacに入れて、ローカルで使うツールです。

また、AIが勝手にコードをマージするものではありません。  
最後に確認して判断するのは人間です。

このプロジェクトは Anthropic 公式プロジェクトではありません。
